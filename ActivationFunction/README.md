# 🧠 Neural Network From Scratch

Built my first neural network in pure Python — no TensorFlow, no PyTorch.

## 📌 What I Built
A student pass/fail predictor that takes:
- Study hours
- Sleep hours  
- Practice tests

And predicts whether a student will PASS or FAIL.

## 📌 What I Used
- Pure Python
- No external libraries except math

## 📌 Concepts Covered
- Inputs, Weights, Bias
- ReLU Activation Function
- Sigmoid Activation Function
- Hidden Layer + Output Layer
- Forward Propagation

## 📌 How It Works
Input → Hidden Layer (ReLU) → Output Layer (Sigmoid) → PASS/FAIL

## 📌 How To Run
python project1.py
